## 🧪 Project Overview

This project explores the use of **Enhanced Rock Weathering (ERW)** as a carbon dioxide removal (CDR) strategy. ERW involves spreading finely crushed silicate rocks (e.g., basalt) over land to accelerate natural weathering processes that capture atmospheric CO₂.

The goal of this project is to:
- Identify **geographically suitable areas** for ERW deployment
- Analyze **environmental variables** such as rainfall, temperature, and soil properties
- Support **decision-making for carbon removal projects**
