# SDG 01: No Poverty

## 🧭 Overview
SDG 01 aims to end poverty in all its forms everywhere by ensuring access to basic services, ownership of land and resources, and resilience to climate-related events.

## 🌐 Geospatial Perspective
Geospatial technologies help identify poverty hotspots, map underserved communities, and guide targeted interventions through spatial data analysis.
